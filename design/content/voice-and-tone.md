## Voice and tone

UI chrome from `content/textations/site.md`. Task text lives in `data/days/*.json`.

### Voice

- Short, actionable sentences
- Specific names and keys in `**emphasis**`
- Calm, no hype
- First person optional in task text; UI stays neutral

### UI chrome

| Context | Copy |
|---------|------|
| Page title | Today |
| Closed state | Day is closed |
| Close button | Close day |
| Modal title | Close this day? |
| Modal body | `{n} open tasks will move to tomorrow.` |
| Confirm | Close day |
| Cancel | Keep working |
| Carryover | From {date} |
| Empty day | No tasks yet. |
| Add task | Add task |
| Add task placeholder | What needs doing? |

### Avoid

- "Todo", "inbox", "productivity"
- Em dashes in new copy
- Generic "Get things done" phrasing
